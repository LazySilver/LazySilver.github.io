二.段落

1、换行

介绍

  1. 按一下enter是宽换行
  2. 按shift+enter是窄换行
  3. 在有序号的情况下窄换行可以续上在上一个序号里，只有宽换行能加上序号
  4. 按shift+tab可以往左一级缩进

三.给字体加效果

1、字体

语法

  • 粗体：** 文字 ** （为了演示以下都加了空格，实际不用加！！！）

  • 删除线：~~ 文字 ~~

  • 下划线: <u> fjskf </u> (建议使用Ctrl+u，可以一开始就按)

  • 斜体：* 文字 *

  • 高亮：== 文字 == （建议要打高亮的先打好 == ，不然后续再回来打很麻烦）
